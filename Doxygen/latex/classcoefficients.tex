\subsection{Detailed Description}
container for action parameters 

All Fermi\+Q\+CD actions are classes and share the same prototype. Parameters are passed to the action via coefficients objects which are nothing more than hash tables.

Example\+: \begin{DoxyVerb}///    gauge_field U(lattice,nc);
///    coefficients gauge;
///    gauge["beta"]=6.0;
///    WilsonGaugeAction::heatbath(U,gauge);
/// \end{DoxyVerb}
 Please check the spelling of the variables you store into the coefficients object (each action has its own coefficients).

Why? This allows the creating of new actions while reusing inverters and simplify passing parameters to the action.
