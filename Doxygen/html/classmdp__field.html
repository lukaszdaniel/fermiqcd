Detailed Description

template<class T>
class mdp_field< T >

most generic field object

Example: 

///    int box[]={10,10,10};
///    mdp_lattice lattice(3,box);
///    mdp_field<float> psi(lattice,10);
///    mdp_site x(lattice);
///    forallsites(x)
///      for(int i=0; i<10; i++)
///         psi(x,i)=0.0;
///    psi.update(); // synchronization
///    psi.save("myfield");
///    psi.load("myfield");
///

Best way to save a field to a VTK file.

Parameters
filenameString giving name of file to create
tInt that specifies which timeslice to save. Value of -1 saves all timeslices.
componentInt specifying which component to save. Value of -1 saves all components. Int naming the ID of processor that should perform the save
ASCIIBool flag to save data in ASCII when true, binary when false
Returns
Returns true on success, throws an error for unexpected conditions.
