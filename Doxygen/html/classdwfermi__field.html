Detailed Description

domain wall fermionic field

Example: 

/// int L5=10; // size in 5th dimension 
/// fermi_field psi(lattice,L5,nc);
/// mdp_site x(lattice);
/// forallsites(x)
///    for(int k=0; k<L5; k++)
///      for(int spin=0; spin<4; spin++)
///        for(int i=0; i<nc; i++)
///          psi(x,k,spin,i)=0.0+0.0*I;
///
