Detailed Description

the gauge field for any SU(n)

Example: 

///    int nc=3; 
///    int box[]={10,8,8,8};
///    mdp_lattice lattice(4,box);
///    gauge_field U(lattice,nc);
///    mdp_site x(lattice);
///    // set_cold(U);
///    forallsites(x)
///       for(int mu=0; mu<U.ndim; mu++)
///          U(x,mu)=1;
///    U.update(); // synchronization
///    U.save("myfield");
///    U.load("myfield");
///

Note that U(x,mu) is and it is a color matrix in SU(nc). is the lattice spacing.
